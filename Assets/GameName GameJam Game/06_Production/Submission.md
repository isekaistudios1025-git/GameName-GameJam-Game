# Submission

## Overview

Tracks requirements for game jam submission.

---

## Requirements

- Playable build
- Windows or Web version
- Game page on itch.io

---

## Checklist

- Game runs without errors
- Controls are explained
- Core loop is complete
- Build is uploaded
- Page is filled out

---

## Final Steps

1. Build game
2. Upload to itch.io
3. Fill out submission page
4. Submit before deadline

---

## Notes

> “A finished small game beats an unfinished big one.”
